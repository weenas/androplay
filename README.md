# AndroPlay 项目

## 项目概述
AndroPlay 是一款专为安卓电视设计的应用程序，支持 Apple AirPlay 功能。用户可以通过苹果设备（iPhone/iPad）将音乐或视频无线投屏到电视上，并在播放音乐时同步显示歌词。

## 核心功能
- **AirPlay 投屏**: 支持从苹果设备无线投屏音频和视频内容
- **歌词同步显示**: 播放音乐时在电视屏幕上同步显示歌词
- **电视端播放控制**: 提供电视端的播放、暂停、进度控制

## 技术要求
- Android TV 应用开发
- AirPlay 协议实现或集成
- 音频/视频解码支持
- 歌词获取与同步显示功能
- 网络传输优化

## 开发计划
1. 需求分析与技术调研
2. 界面设计与用户体验规划
3. AirPlay 协议集成
4. 媒体播放功能开发
5. 歌词同步功能实现
6. 测试与优化
7. 发布准备

## 项目结构
- `app/` - 主应用代码
- `airplay/` - AirPlay 协议相关实现
- `player/` - 媒体播放器组件
- `lyrics/` - 歌词处理模块
- `ui/` - 用户界面组件
